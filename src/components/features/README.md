# Components Features

Chứa các components đặc thù cho từng feature:
- AppointmentCalendar (lịch đặt hẹn)
- ChatBox (chat widget)
- ChecklistForm (form checklist bảo dưỡng)
- FileUploader (upload ảnh/video)
- InvoiceCard (hiển thị hóa đơn)
- NotificationList (danh sách thông báo)
- RatingStars (đánh giá sao)
- VehicleCard (thẻ thông tin xe)
