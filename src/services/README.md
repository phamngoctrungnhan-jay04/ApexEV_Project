# Services

Chứa các service để gọi API:
- authService.js (login, register, logout)
- customerService.js (API cho khách hàng)
- appointmentService.js (đặt lịch, quản lý lịch hẹn)
- maintenanceService.js (bảo dưỡng, checklist)
- invoiceService.js (hóa đơn, thanh toán)
- userService.js (quản lý user, profile)
- partsService.js (phụ tùng)
- awsS3Service.js (upload file lên S3)
- notificationService.js (thông báo)
